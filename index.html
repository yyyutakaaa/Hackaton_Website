🎮 De Complete Escape Room Ervaring

De Cube of Secrets is een innovatieve escape room die fysieke interactie combineert met digitale technologie. Spelers moeten samenwerken om een reeks uitdagingen op te lossen waarbij timing, logisch denken en teamwork essentieel zijn.

Het spel begint wanneer een speler de start knop indrukt. Vanaf dat moment start een 7-segment display een nauwkeurige timer die bijhoudt hoeveel tijd het team nodig heeft om alle puzzels op te lossen. Deze tijdsdruk zorgt voor spanning en moedigt efficiënte samenwerking aan.

Het hart van de uitdaging draait om kleurherkenning en morse code decodering. Via het LCD display krijgen spelers vragen over vlaggen van verschillende landen. Elk antwoord correspondeert met een specifieke kleur die overeenkomt met fysieke arcade drukknoppen en bijpassende kleurenkaarten.

⏱️ Stap 1: Spel Initialisatie

Bij het indrukken van de start knop initialiseert het Arduino systeem alle componenten. De 7-segment display begint te tellen, het LCD toont de eerste vraag, en het systeem genereert willekeurige morse code getallen voor elke kleur. Deze randomisatie zorgt ervoor dat elke speelsessie uniek is en dat spelers het spel meerdere keren kunnen spelen zonder dat ze de oplossing onthouden.

❓ Stap 2: Vraag-Antwoord Mechanisme

Het LCD display toont vragen over vlaggen van landen zoals "Vlag van Frankrijk?" met een scroll-functie voor lange teksten. Spelers moeten de kleuren van de vlag identificeren en in de juiste volgorde bepalen. Een dedicated knop laat spelers naar de volgende vraag springen, wat strategische keuzes mogelijk maakt over welke vragen ze eerst willen beantwoorden.

🎨 Stap 3: Dual-Authentication Systeem

Voor elke kleur moeten spelers een dubbele authenticatie uitvoeren: tegelijkertijd de juiste kleurenkaart voor de TCS230 sensor houden EN de overeenkomstige arcade knop indrukken. Dit mechanisme voorkomt dat spelers gewoon willekeurig knoppen indrukken en vereist echte samenwerking tussen teamleden.

🔢 Stap 4: Morse Code Generatie

Wanneer de juiste combinatie wordt uitgevoerd, speelt de buzzer een uniek getal in internationale morse code af. Elke kleur heeft een willekeurig gegenereerd getal tussen 0-9. Spelers moeten deze codes decoderen en onthouden. De morse implementatie gebruikt officiële timings: punten duren 250ms, strepen 750ms, met correcte pauzes tussen elementen.

📱 Stap 5: Finale Code Validatie

Met de IR afstandsbediening voeren spelers de gedecodeerde getallen in. Het systeem valideert niet alleen de getallen zelf, maar houdt ook bij hoeveel correcte getallen zijn ingevoerd. Wanneer alle drie de vereiste getallen correct zijn ingevoerd, activeert het systeem een succes signaal en stopt de timer. De volgorde waarin getallen ingevoerd worden is bepaald door de antwoorden op de LCD vragen.

💡 Morse Code Uitleg

In morse code wordt elk cijfer voorgesteld door een combinatie van punten (.) en strepen (-). Een punt duurt 250ms, een streep 750ms. Tussen elk punt of streep zit een pauze van 350ms, en aan het einde van elk cijfer een pauze van 500ms.

Technische Componenten & Specificaties

📺

16x2 Character LCD Display

Het primaire interface element dat vragen toont en spelers begeleidt door de escape room ervaring

Technische Specificaties:

  • Pinout: RS=12, Enable=11, D4=5, D5=4, D6=3, D7=2
  • Protocol: 4-bit parallel communicatie voor efficiëntie
  • Library: LiquidCrystal.h (Arduino native)
  • Features: Custom scroll functie voor teksten langer dan 16 karakters
  • Voeding: 5V DC, 2mA typisch verbruik

Implementatie Details:

De scrollTextRight() functie analyseert tekstlengte en past automatisch de weergave aan. Voor teksten korter dan 16 karakters wordt directe weergave gebruikt, voor langere teksten een smooth scroll animatie met 300ms delay tussen frames.

🔢

7-Segment Display Timer

Precise tijdmeting systeem dat de totale speelduur bijhoudt en competitieve elementen toevoegt

Technische Specificaties:

  • Controle Pin: Pin 52 (digitaal output voor einde signaal)
  • Functionaliteit: Automatische start bij begin spel
  • Precisie: Milliseconde nauwkeurigheid
  • Stop Conditie: Geactiveerd wanneer alle puzzels opgelost zijn

Game Design Functie:

De timer creëert urgentie en competitie tussen teams. Het systeem registreert digitalWrite(52, HIGH) wanneer de variabele z == 3, wat betekent dat alle drie vereiste morse codes correct zijn ingevoerd.

🎨

TCS230 RGB Kleurensensor

Geavanceerde kleurdetectie sensor die lichtfrequenties converteert naar digitale RGB waarden

Technische Specificaties:

  • Pinout: S0=22, S1=24, S2=26, S3=28, Output=30
  • Technologie: Fotodiode array met RGB filters
  • Frequentieschaal: 20% (S0=HIGH, S1=LOW) voor optimale nauwkeurigheid
  • Meetbereik: Detecteert 5 verschillende kleuren met tolerantiewaarden
  • Detectie Methode: pulseIn() functie meet frequentie output

Kalibratie Waarden:

Na uitgebreide testing zijn specifieke RGB ranges bepaald: Blauw (50-80, 35-65, 25-45), Rood (20-50, 35-70, 40-60), Groen (40-60, 30-50, 35-55), Geel (10-35, 15-40, 30-50), Wit (<25, <25, <25). Deze waarden compenseren voor omgevingslicht en materiaal reflectie.

🔴

Arcade Button Interface

Professionele arcade-style drukknoppen met ingebouwde verlichting voor tactile feedback

Technische Specificaties:

  • Kleur Mapping: Rood (A1), Groen (A4), Geel (A2), Blauw (A0), Wit (A3)
  • Aanvullende Knoppen: Start knop (A7), Volgende vraag knop
  • Input Type: Analoge pinnen gebruikt als digitale inputs
  • Pull-up Configuratie: Interne pull-up resistors geactiveerd
  • Debouncing: Software filtering voor stabiele input

Logica Implementatie:

Elke knop vereist drie condities: digitalRead(knop) == HIGH (fysieke druk), detectColor() == "Kleur" (sensor match), en KleurInAntwoord("Kleur", antwoorden) (validatie tegen huidige vraag). Deze triple-check voorkomt false positives.

🔊

Morse Code Audio System

Precisie buzzer systeem dat internationale morse code standaarden implementeert

Technische Specificaties:

  • Pin Assignment: Pin 9 (PWM capable voor tone generatie)
  • Frequentie: 1000Hz carrier wave voor optimale hoorbaarheid
  • Timing Standaarden: DOT=250ms, DASH=750ms, DSPACE=350ms, LSPACE=500ms
  • Implementatie: Function pointer array voor efficiënte code organisatie

Morse Code Mapping:

Volledig cijfer set 0-9 geïmplementeerd volgens ITU-R M.1677-1 standaard. Elke morse functie (morse_0 tot morse_9) gebruikt de gedefinieerde timing constanten voor consistente audio output. De morseGetallen[] array gebruikt function pointers voor directe indexing.

📱

IR Communication System

Draadloos input systeem voor finale code invoer met NEC protocol ondersteuning

Technische Specificaties:

  • Receiver Pin: Pin 8 (interrupt capable voor real-time decoding)
  • Library: IRremote.h met LED feedback ondersteuning
  • Protocol: NEC infrared protocol (38kHz carrier)
  • Timeout: 8 seconde restart timer voor betrouwbare operatie
  • Validation: Hexadecimale code matching tegen lookup table

Code Validation Logic:

Ontvangen IR codes worden gematcht tegen de hexCombo[] array met bekende remote button waarden. De IndexVan() functie vindt de numerieke waarde, ZitInArray() controleert of deze in de correcte set zit. Success counter z wordt geïncrementeerd bij matches.

🏗️ Systeem Architectuur

De Cube of Secrets gebruikt een modulaire architectuur waarbij elk component een specifieke rol vervult in de game loop. Het Arduino Uno fungeert als centrale verwerkingseenheid die alle inputs en outputs coördineert.

Input Layer

TCS230 sensor + Arcade buttons + IR receiver

→

Processing Layer

Arduino Uno met custom game logic

→

Output Layer

LCD display + Buzzer + 7-segment timer

De software architectuur gebruikt event-driven programming waarbij elk component reageert op specifieke triggers. Random number generation zorgt voor replay value, terwijl state management ervoor zorgt dat het spel logisch verloopt van start tot finish.
