How It Works

1

Start the Challenge

Press the start button to begin your escape room adventure. A 7-segment display will start tracking your time as you work to solve the puzzle.

2

Read the Clues

Questions appear on the LCD display. Each question has a color-based answer that corresponds to the physical colored buttons and cards.

3

Use Color Cards

Hold the matching color card in front of the color sensor while pressing the corresponding button to receive Morse code numbers through the buzzer.

4

Enter the Code

Use the IR remote control to enter the numbers in the correct order determined by the LCD questions. Solve all puzzles to escape!
