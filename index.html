How It Works

  1. Press the start button to activate the timer on the 7-segment display.
  2. Answer questions on the LCD display. Each answer is a color.
  3. Hold the matching color card in front of the color sensor while pressing the corresponding button.
  4. Listen to the Morse code number from the buzzer.
  5. Enter the numbers using the IR remote in the correct order.

Replayability

Five colored buttons allow endless possibilities. Not every button is needed each time, so every game is unique!
